# E-Commerce-Sales-Supply-Chain-Analysis-Using-Kaggle-SQL-Python-Power-BI
End-to-end analysis of an e-commerce dataset to understand sales performance, product trends, customer behavior, and delivery efficiency. The analysis is done using Python for data cleaning, SQL for querying, and Power BI for visualization.  The dataset used is the Brazilian E-Commerce Public Dataset (Olist) from Kaggle.

Objectives

•	Analyze overall sales performance

•	Identify top product categories by revenue

•	Study order delivery time and delays

•	Build an interactive dashboard for business insights
